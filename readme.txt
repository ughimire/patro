Just another Nepali calender plugin. Simple nepali calender.

== Description ==

This is a simple Nepali calender plugin plus calender widget.
Soon we are going to develop it to Nepali event registration calender plugin.



== Installation ==

1. Upload  whole `Patro` folders file  to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add nepali calender widget from appearence menu.
1. You can add short code [nepali-patro] on post or pages.
